  • Story

    I've decided to put some effort into Cloud DevOps path and build a running solution on AWS:

    • Started from setting up Kubernetes cluster on Amazon EKS with EKSCTL. Managed to tweak settings to stay within Free Tier on AWS.
    • Next evening CI/CD pipeline on Jenkins
      • triggered by github commit with Jenkins webhook
      • building docker image with python webapp
      • running the webapp with docker-compose on the same machine as Jenkins
    • TODO: deploy to kubernetes - prepare actual deployment, with replicas + distinguish between replicas by displaying hostname

  • CI/CD Pipeline

    git commit; git push ↓
    GitHub calls /webhook API on Jenkins in AWS ↓
    Jenkins executes Preparation step: checkout from github ↓
    Jenkins executes Build step: docker build ↓
    Jenkins executes Deploy step: docker-compose up
